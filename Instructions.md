# Instructions for Use

## Setup
1. Copy the full `data:text/html` URI provided in the README.
2. Open your browser (Chrome, Safari, or Firefox).
3. Paste the URI into the address bar and press Enter.
4. **Tip:** Bookmark this page or "Add to Home Screen" on mobile for instant access.

## Formatting Tools
- **B / I / U / S**: Standard character formatting.
- **List**: Creates bulleted lists.
- **H1 / H2**: Creates large and medium headings.
- **Para**: Resets the current line to a standard paragraph.

## Copying Content
- **Copy MD**: This is best for **Google Keep** or **GitHub**. It converts your styles into symbols (like `**bold**`).
- **Copy Plain**: Use this to strip all styles and symbols, leaving only the words.

## Important Note
**Pro Editor does not save data.** If you refresh the page or close the tab, your text will be lost. Always click a "Copy" button before leaving the page.
