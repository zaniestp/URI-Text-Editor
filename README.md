A professional, zero-dependency Markdown and Plain Text editor that lives entirely within a single Browser URI. No hosting required.

## ✨ Features
- **Rich Text Editing:** Bold, Italic, Underline, Strikethrough, Lists, and Headings.
- **Markdown Conversion:** Instantly convert formatted text to Markdown for Google Keep, GitHub, or Reddit.
- **Plain Text Mode:** Strip all formatting for clean text copying.
- **Mobile First:** Designed with a sticky header and scrollable toolbar for easy thumb access.
- **Zero Storage:** Privacy-focused; data is never saved to a server or local storage.

## 🛠️ Usage
1. Open the URI.
2. Type or paste your content.
3. Highlight text and use the toolbar to format.
4. Click **Copy MD** to get Markdown or **Copy Plain** for raw text.
